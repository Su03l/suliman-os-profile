# SulimanOS - معرض أعمال بواجهة مستوحاة من نظام التشغيل Windows 10

هذا المشروع عبارة عن معرض أعمال تفاعلي مصمم ليبدو وكأنه سطح مكتب لنظام التشغيل Windows 10. يعرض المشاريع والمهارات والخبرات الخاصة بـ سليمان يوسف، وهو مهندس برمجيات ومطور Full-Stack. تم بناء الواجهة باستخدام React و Tailwind CSS، وتتميز بنافذة قابلة للسحب وشريط مهام وشاشة تمهيد، مما يخلق تجربة مستخدم فريدة وجذابة.

## الميزات

*   **واجهة مستوحاة من نظام التشغيل Windows 10:** واجهة سطح مكتب حديثة ومألوفة.
*   **نوافذ قابلة للسحب:** فتح وتحريك النوافذ في جميع أنحاء سطح المكتب.
*   **شريط المهام:** إدارة التطبيقات المفتوحة والوصول إلى قائمة ابدأ.
*   **شاشة التمهيد:** محاكاة لتسلسل بدء التشغيل لإضفاء طابع واقعي.
*   **تصميم متجاوب:** تم تصميم معرض الأعمال للعمل على مختلف أحجام الشاشات.
*   **محتوى تفاعلي:** استكشف المشاريع والمهارات والخبرات بطريقة جذابة.

## التقنيات المستخدمة

تم بناء هذا المشروع باستخدام:

*   **Vite:** أداة تطوير للواجهة الأمامية من الجيل التالي تتميز بالسرعة والخفة.
*   **React:** مكتبة JavaScript لبناء واجهات المستخدم.
*   **TypeScript:** مجموعة شاملة من JavaScript يمكن تحويلها إلى JavaScript عادي.
*   **Tailwind CSS:** إطار عمل CSS يعتمد على الأدوات المساعدة لتطوير واجهة المستخدم بسرعة.
*   **Framer Motion:** مكتبة حركة جاهزة للإنتاج لـ React.
*   **Zustand:** حل صغير وسريع وقابل للتطوير لإدارة الحالة.
*   **shadcn-ui:** مجموعة من المكونات القابلة لإعادة الاستخدام يمكنك نسخها ولصقها في تطبيقاتك.
